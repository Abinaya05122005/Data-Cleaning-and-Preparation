# Data Cleaning & Preparation – Project 1

## Intern Details 

**Project:** Data Cleaning & Preparation
**Organization:** DecodeLabs
**Batch:** 2026

## Project Objective

The objective of this project is to clean a raw dataset by identifying missing values, checking duplicate records, and verifying data formats to prepare the dataset for further analysis.

## Dataset

**File Name:** Dataset for Data Analytics.xlsx

## Tasks Performed

* Loaded the dataset using Python (Pandas).
* Identified missing values in the dataset.
* Filled missing values in the **CouponCode** column with **"No Coupon"**.
* Checked for duplicate rows and verified that no duplicate records were present.
* Verified that all **OrderID** values are unique.
* Verified and standardized the **Date** column using `pd.to_datetime()`.
* Verified numeric and text data types.
* Exported the cleaned dataset as **cleaned_Dataset_for_Data_Analytics.xlsx**.
